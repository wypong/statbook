    Section 6.1 Sample Spaces and Events

    A random experiment is a process with a known collection of possible outcomes, but the outcome of one trial is not known ahead of time. Flipping a coin, rolling a die, and selecting a student at random are all random experiments.
    🔗
    The set of all possible outcomes is called the sample space, usually written \(S\text{.}\) An event is a subset of the sample space.
    🔗

    Example 6.1.1. A Sample Space for a Die Roll.

    If a fair six-sided die is rolled once, then the sample space is
    🔗
    \begin{equation*} S=\{1,2,3,4,5,6\}. \end{equation*}
    The event “an even number occurs” is
    🔗
    \begin{equation*} E=\{2,4,6\}, \end{equation*}
    and the event “a number greater than 4 occurs” is
    🔗
    \begin{equation*} G=\{5,6\}. \end{equation*}
    🔗
    Once we treat events as sets, we can use ordinary set operations.
    🔗
    The complement of an event \(A\text{,}\) written \(A^c\text{,}\) consists of all outcomes in the sample space that are not in \(A\text{.}\)
    🔗
    The union \(A\cup B\) consists of outcomes that are in \(A\) or in \(B\) or in both. The intersection \(A\cap B\) consists of outcomes that are in both \(A\) and \(B\text{.}\)
    🔗

    Example 6.1.2. Set Operations with Two Coin Flips.

    If a coin is flipped twice, then the sample space is
    🔗
    \begin{equation*} S=\{HH,HT,TH,TT\}. \end{equation*}
    Let \(A\) be the event “exactly one head occurs,” so \(A=\{HT,TH\}\text{.}\) Let \(B\) be the event “the first flip is a head,” so \(B=\{HH,HT\}\text{.}\)
    🔗
    Then
    🔗
    \begin{equation*} A^c=\{HH,TT\},\qquad A\cup B=\{HH,HT,TH\},\qquad A\cap B=\{HT\}. \end{equation*}
    🔗
    Two events are mutually exclusive if they cannot happen at the same time. In set language, that means their intersection is empty.
    🔗

    Example 6.1.3. Mutually Exclusive Events.

    When a die is rolled, the event “the outcome is 2” and the event “the outcome is 5” are mutually exclusive, because no single roll can be both 2 and 5.
    🔗
    On the other hand, the events “the outcome is even” and “the outcome is greater than 3” are not mutually exclusive, because 4 and 6 belong to both events.
    🔗
    🔗

    Activity 6.1.1. Sample Spaces and Events.

    Practice writing sample spaces and identifying events for simple random experiments.
    🔗

    (a)

    Write the sample space for one roll of a fair six-sided die. Then list the outcomes for the event “the number is at least 4.”
    🔗
    🔗

    (b)

    Write the sample space for one coin flip and list the outcomes for the event “heads occurs.” Then do the same for the event “tails occurs.”
    🔗
    🔗

    (c)

    For two coin flips, write the event “exactly one head occurs” as a set of outcomes.
    🔗
    🔗
    🔗

    Activity 6.1.2. Complements, Unions, and Intersections.

    Use the language of sets to describe common probability events.
    🔗

    (a)

    Using the sample space for one die roll, let \(A\) be the event “the outcome is even” and let \(B\) be the event “the outcome is greater than 3.” Write \(A^c\text{,}\) \(A\cup B\text{,}\) and \(A\cap B\text{.}\)
    🔗
    🔗

    (b)

    For two coin flips, let \(A\) be the event “the first flip is a head” and let \(B\) be the event “exactly one head occurs.” Find \(A\cap B\text{.}\)
    🔗
    🔗

    (c)

    Decide whether the events “the number is 1” and “the number is 6” are mutually exclusive. Explain your answer.
