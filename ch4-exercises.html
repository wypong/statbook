    Exercises 4.4 Exercises

    Exercise Group.

    For Exercises 1-4, find the requested quartiles, IQRs, or fences.
    🔗

    1.

    Find \(Q_1\text{,}\) \(Q_2\text{,}\) and \(Q_3\) for 2, 4, 5, 7, 8, 10, 12, 14, 18.
    🔗
    🔗

    2.

    Find the quartiles and IQR for 6, 7, 9, 10, 12, 14, 16, 18.
    🔗
    🔗

    3.

    The five-number summary of a data set is 3, 8, 11, 17, 24. What is the IQR?
    🔗
    🔗

    4.

    A data set has \(Q_1=15\) and \(Q_3=27\text{.}\) Find the 1.5 IQR fences.
    🔗
    🔗
    🔗

    5.

    A data set has \(Q_1=10\text{,}\) \(Q_3=18\text{,}\) and includes 1, 6, 12, 15, 19, and 33. Which values are possible outliers by the 1.5 IQR rule?
    🔗
    🔗

    6.

    Explain in words what the interquartile range tells you about a distribution.
    🔗
    🔗

    7.

    Two data sets both have mean 20. One has standard deviation 2 and the other has standard deviation 9. Which data set is more spread out?
    🔗
    🔗

    8.

    For the population data set 2, 4, 6, 8, compute the population mean, the population variance, and the population standard deviation.
    🔗
    🔗

    9.

    For the sample data set 5, 7, 9, 11, compute the sample mean and the sample standard deviation.
    🔗
    🔗

    10.

    A data set has mean 50 and standard deviation 8. Find the z-score of 62.
    🔗
    🔗

    11.

    A data value has z-score \(-1.5\text{.}\) Explain what this tells you about the value relative to the mean.
    🔗
    🔗

    12.

    Machine A has mean 40 and standard deviation 4. Machine B has mean 120 and standard deviation 6. Find the relative standard deviation for each machine and decide which machine has greater spread relative to its mean.
    🔗
