    Appendix C Normal Distribution

    This appendix gives the standard normal cumulative distribution table. Each entry is \(P(Z\le z)\) for a standard normal random variable \(Z\sim N(0,1)\text{.}\)
    🔗
    The table lists only nonnegative \(z\)-values. For negative values, use symmetry: \(P(Z\le -z)=1-P(Z\le z)\text{.}\)
    🔗
    To keep the print layout readable, the columns are split into two companion tables.
