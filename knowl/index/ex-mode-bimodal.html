Example 3.1.3. One Mode, Two Modes, or No Mode.

In the data set 2, 3, 3, 4, 6, the mode is 3 because it appears most often. In the data set red, blue, blue, green, green, there are two modes, blue and green, so the distribution is bimodal. In the data set 1, 2, 3, 4, each value appears once, so by convention the data set has no mode.
